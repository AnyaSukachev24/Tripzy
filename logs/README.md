# 📊 Tripzy Logs

This folder contains all application and server logs for the Tripzy project.

## Log Files

- **`server.log`** - Main server log
- **`server_*.log`** - Numbered server logs from different runs
- **`debug.log`** - Debug output and troubleshooting information
- **`verify.log`** - Verification and validation logs

## Usage

These logs are useful for:
- Debugging issues
- Monitoring server performance
- Tracking API calls and responses
- Understanding system behavior

## Maintenance

Consider periodically archiving or removing old log files to keep this directory clean.
